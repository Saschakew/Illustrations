Welcome to the SVAR Visualization Template

This is a template for building interactive visualizations for Structural Vector Autoregression (SVAR) models. It provides a starting point for demonstrating complex econometric concepts in an intuitive, hands-on way.

Using this template, you can allow users to:

  • Generate data from various data-generating processes.
  • Explore different identification and estimation strategies.
  • Visualize the impact of parameter choices on model outcomes.
  • Understand the underlying mathematics through dynamically updated formulas.

Tweak a control and watch plots, matrices, and LaTeX formulas update in real time.
